# craftworks-english-lab

TOEIC Part 5（文法・語彙）対策用の Web アプリです。  
カテゴリ選択 → クイズ出題 → 解説表示 を目的としています。

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Features (WIP)
- カテゴリ選択（8カテゴリ）
- 選択カテゴリのみ出題
- 問題データはカテゴリ別JSONで管理

## Development
```bash
npm install
npm run dev
